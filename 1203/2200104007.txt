クラス C とクラス W の 2 クラス分類問題を考える．
このとき，クラス C と判別されたデータについて，実際にクラス C であったデータの数を TP，実際はクラス W であったデータの数を FP とする．
また，クラス W と判別されたデータについて，実際にクラス W であったデータの数を TN，実際はクラス C であったデータの数を FN とする．
ここでクラス C と判別した中で実際にクラス C であった割合を表す精度 (Accuracy) は，Accuracy = TP / (TP + FP) であり，クラス C の中で実際にクラス C と判別された割合を表す再現率 (Recall) は，Recall = TP / (TP + FN) である．
これら 2 つの指標の調和平均を算出した値が F 値であり，精度と再現率を総合した指標である．
その算出の過程で算術平均ではなく調和平均が用いられる理由は，精度と再現率の算出過程における分子がそれぞれ等しく，分母が異なるためである．
